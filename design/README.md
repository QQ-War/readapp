# iOS 风格界面效果图

下面的 SVG 对应最新版 iOS 风格稿，可在没有 Android 预览环境时快速理解新版界面层级、留白与配色：

- `ios_style_bookshelf.svg`：深色书架页，顶部设定入口 + 右上折叠菜单，主区仅保留搜索与卡片式书籍条目。
- `ios_style_reader.svg`：阅读/播放页，目录折叠在“目录”按钮，段落高亮仅保留当前/预读，支持上下段切换与点击空白处收起控件。
- `ios_style_settings.svg`：设置页，服务器地址与账号同权重，并补充 TTS 引擎选择与 TTS 预读段落数调节。

SVG 可直接在浏览器打开或导入到设计工具（如 Figma）继续迭代，也可以作为 Android Compose 布局的快速参照。
